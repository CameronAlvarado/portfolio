Key Conservation

A mobile app designed to connect supporters to animal conservationists from all around the globe. Conservationists can create profiles and post content pertaining to their campaigns. Supporters can sign up and find, follow, and in the future donate to animal conservation groups.

React Native | JavaScript | Node.js

  • Functioned remotely on a cross-functional team of 8, composed of 2 UX Designers and 6 Full-Stack Developers.
  • Built on a pre-existing codebase to create new conservationist onboarding process and add map/location functionality.
  • Integrated Airtable API for new conservationist applicants, allowing easier stakeholder review and access permission for new conservationist applicants.

Digital Wasteland

A text-based RPG game made during a week long hack-a-thon. Users can register an account, explore 120 generated rooms and pick up items throughout the wasteland. They can also chat with other users using the Pusher ChatKit API. I worked mostly on the front-end and integrated React Vis and CSS.

Javascript | React | Python | Django

  • Utilized Python / Django to build a back-end server.
  • Used React to build a front-end interface
  • Integrated Pusher to enable chat between online users
Github Repo

Silent Auction

A simplified online auction app that allows sellers and buyers to register and sell/bid on items. Sellers can upload images, add descriptions and set duration for items. Buyers can make bids and outbid other buyers.

Javascript | React | Node.js

  • Programmed the front-end, assembled state management and solved Javascript logic.
  • Applied authentication for protected routes.
  • Utilized Axios to make API calls to back-end servers.
Github Repo

Javascript | React | Redux

  • Utilized Axios to call API.
  • Loads Youtube video for each flight.
